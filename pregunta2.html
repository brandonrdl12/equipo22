¿Para que sirve un CSS?


CSS es un lenguaje utilizado en la presentación de documentos HTML. Un documento HTML viene siendo coloquialmente “una página web”.

Entonces podemos decir que el lenguaje CSS sirve para organizar la presentación y aspecto de una página web.
